Proyectos Académicos

HG Coffee (Catálogo de Productos)

Tecnologías: PHP, MySQL

Este proyecto consistió en el desarrollo de una aplicación web para la gestión de productos de una cafetería. Se implementó un catálogo digital que permitía registrar, actualizar y eliminar artículos, mostrando la información al usuario en tiempo real.

Aprendizajes principales

  • Uso de PHP para desarrollo backend
  • Conexión a bases de datos MySQL
  • Manejo de operaciones CRUD (crear, leer, actualizar, eliminar)

Sistema Gestor de Control de Asistencia

Tecnologías: PHP, MySQL, HTML, CSS, JavaScript

Desarrollo de una aplicación web para la empresa HERCEMA, dedicada a la compra-venta de ganado. El sistema automatizó el registro de asistencia de empleados mediante DNI, reduciendo errores y mejorando la puntualidad.

Alcance: Control de asistencias por turnos, generación de reportes, administración de usuarios y cargos.

Aprendizajes

  • Aplicación de metodologías ágiles (XP)
  • Desarrollo full stack
  • Despliegue en entorno real y capacitación a usuarios finales

SkinSpectrum - Detección Temprana de Cáncer de Piel

Tecnologías: Flutter, Firebase, Firestore, Vertex AI (CNN), Google Cloud Functions, Astro

Plataforma integral que conecta pacientes y médicos para la detección temprana de cáncer de piel en personas con tonos de piel oscuros.

Incluye:

  • App móvil en Flutter para capturar imágenes de lesiones cutáneas
  • Modelo CNN en Vertex AI entrenado con datasets médicos para clasificar lesiones
  • Base de datos en Firestore para almacenamiento de resultados y gestión de usuarios
  • Web en Astro para que doctores gestionen pacientes, suban resultados y colaboren con diagnósticos

Alcance: Sistema multiplataforma con enfoque social y médico. Los pacientes reciben resultados en la app, mientras que los médicos pueden analizar, registrar y dar seguimiento en la web.

Aprendizajes

  • Integración de inteligencia artificial en aplicaciones móviles
  • Despliegue de modelos en la nube
  • Autenticación segura con Firebase
  • Diseño UI/UX para aplicaciones médicas
